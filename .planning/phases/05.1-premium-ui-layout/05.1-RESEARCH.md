# Phase 5.1: Premium UI Layout & Accessibility (Shopify Inspired) - Research

Bản nghiên cứu này cung cấp các đoạn mã CSS, HTML mẫu và kỹ thuật lập trình Razor để triển khai thiết kế Shopifi-Inspired cùng các tiêu chuẩn tiếp cận WCAG 2.1 AA.

## 1. Nghiên cứu Kỹ thuật Thẩm mỹ (UI/UX)

### 1.1. Bảng màu CSS Custom Variables (`admin-theme.css`)
Hệ thống token màu sắc sẽ được định nghĩa ở cấp độ `:root` để dễ dàng tái sử dụng trên toàn hệ thống:
```css
:root {
    /* Shopifi-Inspired Cream-Mint Canvas Tokens */
    --colors-primary: #000000;
    --colors-ink: #111111;
    --colors-on-primary: #ffffff;
    --colors-canvas-cream: #fbfbf5;
    --colors-canvas-light: #ffffff;
    --colors-aloe-10: #c1fbd4;
    --colors-pistachio-10: #d4f9e0;
    
    /* Muted link colors and borders */
    --colors-shade-30: #e4e4e7;
    --colors-shade-50: #71717a;
    --colors-shade-70: #27272a;
    --colors-hairline: rgba(0, 0, 0, 0.08);

    /* Elevation Shadows (Level 3 Stacked Soft Shadow Stack) */
    --shadow-level3: 
        0 1px 2px rgba(0,0,0,0.02),
        0 2px 4px rgba(0,0,0,0.02),
        0 4px 8px rgba(0,0,0,0.02),
        0 8px 16px rgba(0,0,0,0.02),
        0 0 0 1px rgba(0,0,0,0.04);
}
```

### 1.2. Kính mờ (Glassmorphism) & Mesh Gradient Nền (Login Card)
Nền mesh gradient động tạo ra cảm giác không gian cinematic mượt mà, kết hợp với hộp thẻ kính mờ:
```css
.login-bg {
    background: linear-gradient(135deg, #fbfbf5 0%, #d4f9e0 50%, #c1fbd4 100%);
    background-size: 400% 400%;
    animation: meshMovement 15s ease infinite;
}

@keyframes meshMovement {
    0% { background-position: 0% 50% }
    50% { background-position: 100% 50% }
    100% { background-position: 0% 50% }
}

.login-card-glass {
    background: rgba(255, 255, 255, 0.7);
    backdrop-filter: blur(16px) saturate(140%);
    -webkit-backdrop-filter: blur(16px) saturate(140%);
    border: 1px solid rgba(255, 255, 255, 0.4);
    box-shadow: var(--shadow-level3);
}
```

### 1.3. Sidebar và Điều hướng Responsive
Để Sidebar trượt và co giãn linh hoạt trên mobile mà không dùng Javascript phức tạp, chúng ta tận dụng Grid System của Bootstrap 5 và thuộc tính `@media`:
```css
.sidebar-panel {
    background-color: #0f172a; /* Deep Slate */
    transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.sidebar-panel a {
    color: #94a3b8;
    border-radius: 9999px; /* Pill shape */
    transition: all 0.2s ease;
    margin: 4px 12px;
}

.sidebar-panel a:hover {
    background-color: rgba(255, 255, 255, 0.05);
    color: #ffffff;
}

.sidebar-panel a.active {
    background-color: var(--colors-primary);
    color: var(--colors-on-primary);
}
```

## 2. Nghiên cứu Kỹ thuật Tiếp cận (Accessibility)

### 2.1. Focus Outline tùy biến
Đảm bảo khi người dùng nhấn `Tab`, phần tử đang được focus hiển thị rõ ràng:
```css
a:focus-visible, button:focus-visible, input:focus-visible, select:focus-visible {
    outline: 2px solid #000000 !important;
    outline-offset: 2px;
    box-shadow: 0 0 0 4px rgba(0, 0, 0, 0.15) !important;
}
```

### 2.2. Liên kết Nhãn Razor Helper
Trong ASP.NET Core MVC, việc khai báo `asp-for` trên nhãn sẽ tự động biên dịch sang thẻ HTML đúng chuẩn tiếp cận:
```html
<!-- Razor Code -->
<label asp-for="Username" class="form-label"></label>
<input asp-for="Username" class="form-control" />

<!-- Sinh ra mã HTML -->
<label class="form-label" for="Username">Username</label>
<input class="form-control" id="Username" name="Username" type="text" value="" />
```

## 3. Chuyển đổi Layout Động trong Razor View
Sử dụng mã C# ở đầu View `Details.cshtml` để chọn layout phù hợp với vai trò người xem:
```csharp
@{
    Layout = User.Identity != null && User.Identity.IsAuthenticated 
             ? "_LayoutAdmin" 
             : "_Layout";
    ViewData["Title"] = Model.Title;
}
```
Và trong `PostController.cs`, cấu hình action:
```csharp
[AllowAnonymous]
public IActionResult Details(int id)
{
    var post = _context.Posts.Include(p => p.Category).FirstOrDefault(p => p.Id == id);
    if (post == null) return NotFound();
    return View(post);
}
```
