# Roadmap

**Project:** CMS Full-stack

## Phase 5: Validation & Security
**Goal:** Bảo mật hệ thống quản trị và kiểm soát dữ liệu đầu vào.
**Depends on:** None
**Requirements:** REQ-SEC-01, REQ-SEC-02, REQ-SEC-03

## Phase 6: WebAPI RESTful
**Goal:** Xây dựng các API trả về JSON phục vụ cho ReactJS frontend.
**Depends on:** Phase 5
**Requirements:** REQ-API-01, REQ-API-02, REQ-API-03

## Phase 7: Frontend ReactJS Setup
**Goal:** Khởi tạo dự án và giao diện Frontend bằng ReactJS.
**Depends on:** Phase 6
**Requirements:** REQ-FE-01, REQ-FE-02

## Phase 8: Frontend API Integration
**Goal:** Kết nối ReactJS với WebAPI để lấy dữ liệu.
**Depends on:** Phase 7
**Requirements:** REQ-FE-03, REQ-FE-04

## Phase 9: Routing SPA
**Goal:** Điều hướng giữa các trang trên ReactJS (Trang chủ -> Chi tiết).
**Depends on:** Phase 8
**Requirements:** REQ-FE-05, REQ-FE-06

## Phase 10: gRPC (Advanced)
**Goal:** Ứng dụng gRPC cho các tác vụ giao tiếp tốc độ cao.
**Depends on:** Phase 9
**Requirements:** REQ-ADV-01, REQ-ADV-02

## Phase 11: Debug & Optimization
**Goal:** Sửa lỗi và hoàn thiện ứng dụng trước khi báo cáo.
**Depends on:** Phase 10
**Requirements:** REQ-OPT-01, REQ-OPT-02
